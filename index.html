Implementation of a Geometric Sequence Technique for Effective RTO Estimation in CoAP

Authors : Vishal Rathod and Mohit P. Tahiliani
Github Results


Introduction


A Geometric Sequence Technique (GST) is an effective RTO estimation in Constrained Application Protocol (CoAP). GST retains the previous RTO value after having received the ACK for the retransmitted packet and eventually returns to the default value by decreasing the RTO depending on the number of consecutive successful transmissions.
CoAP uses a fixed RTO value with BEB in case of retransmissions. Two new techniques, namely Fullbackoff1 variant and Fullbackoff2 variant are proposed to improve the RTO estimation in CoAP. In this work, we have implemented an effective backoff mechanism for CoAP which leverages geometric sequences.
GST is a simple enhancement to the Fullbackoff2 variant. Unlike the latter which decreases RTO in a binary exponential manner (i.e., reduces RTO by half) when packets are cleared without retransmissions.
In our work, GST minimizes the Flow Completion Times (FCT), reduces the number retransmissions and improves the network throughput.
